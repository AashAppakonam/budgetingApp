# budgetingApp
Personal budgeting mobile app intended to track daily spending into categories so I don't overspend.


Prompt:

I am trying to build a personal budgeting app. The main purpose of this app is to track daily spending so I don't overspend.

Here's how I would use the app and my rough requirements:

I would open the app at the beginning of the month and enter my total budget (or income). This would update the main screen of the app with a pie chart/circle graphic with the total money I have left for the month. (Ideally, this should be in the color green to indicate a positive addition). At the bottom of the main screen, there should be a log of all my entries. 

There should also be a button that opens another screen to create a new entry. The new entry section should have a section to enter the amount, a name (it should remember previous names so it will autofill after using the app for a while), additional notes, catergory selector, and a submit button. After clicking submit, the app returns to the main screen with the updates pie chart. 

The main screen pie chart should have labels for each category (ideally it should be icons). There should also a button somewhere to add or delete categories. When adding a new category, I should be able to add a name and pick from a group of common icons used in many budeting apps to represent a category (consider using emojis). 

This is the basics of the app for now. Feel free to add or suggest improvements or add certain navigation features, etc. that you see fit. I also would like the app to be mainly dark mode and have minimilistic look and very fluid navigation. 

Please give me the code for a flutte project for this app.
